# Work-Scripts
This is a repository that is going to have all of the scripts that I have created for my jobs
The config.py has been adjusted to remove sensitive path files to work folders, should be replaced with folders dependent on job
The PDF Automation specifically looks for a JHA file and a WO order file as well as the site number, it will auto select the options that it sees
once the user reviews the file they can hit submit if correct, if wrong they can just leave it
If they hit submit it will automatically create folders in the right work folders using the site numbers and move the file into it
