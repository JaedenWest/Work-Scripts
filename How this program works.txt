How this works
1. It pulls from the most recent pdf you downloaded and opens it
2. It will auto find the site number from the file name
3. If it has JHA or WO in its name it will auto select that option
4. The user reviews the JHA or WO and if its good, with approved already selected you can hit submit
5.It will create a folder in the JHA or WO folder
6. The folder with be the site number, it will automatically fill the digits with 0s up to 4 digits and then drag the pdf into it
7. It will say if the file has moved successfully, or if it already existed.
8. If moved successfully you can close app and reopen when next pdf is downloaded
